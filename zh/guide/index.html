# 介绍

警告

这是文档的翻译版本。它可能是不完整或过时的。对于当前版本切换到英文版。

Naninovel是 Unity引擎 的扩展。包含C#框架，以及编辑器程序，用以开发VN视觉小说类游戏.

注意

尽管Naninovel不会限制你想用Unity实现什么，但有部分前提条件（如支持unity版本，项目配置，目标平台）来确保Naninovel内置功能的正常工作。详情参考兼容性页

确保你第一次使用时阅读 快速上手。

如果你想查找某个问题，尝试使用本站上方的搜索栏；或者查阅 常问问题, 这里有一些我们已经回答过的常见问题。所有可用脚本命令，支持参数，已经可用示例都列举在 API参考。如果找不到所需问题也可以直接联系技术支持 。
